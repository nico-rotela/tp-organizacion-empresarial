## Escenario elegido

Escenario B – Análisis de Ventas de una Pequeña Empresa.

## Descripción del proyecto

El proyecto consiste en analizar un conjunto de datos de ventas utilizando Python, Git, GitHub y Jira Software como herramientas de organización y control de versiones.

Se desarrolló un script capaz de:

* calcular ventas totales,
* identificar el producto más vendido,
* generar un gráfico simple de ventas por producto.

## Estructura del repositorio

```bash
datos/
scripts/
resultados/
```

## Dataset utilizado

Se utilizó un dataset simple en formato CSV, almacenado en la carpeta `/datos`.

## Tecnologías utilizadas

* Python
* Pandas
* Matplotlib
* Git
* GitHub
* Jira Software

## Ejecución del proyecto

Ejecutar el script:

```bash
python analisis_ventas.py
```

El gráfico generado se almacena en la carpeta `/resultados`.

## Organización del trabajo

El proyecto fue desarrollado individualmente. Para cumplir con la metodología de trabajo colaborativo propuesta en la consigna, se simularon distintos roles de la célula de desarrollo mediante tareas diferenciadas en Jira.

Cada Issue ID representa una responsabilidad específica dentro del flujo de trabajo y se vinculó con los commits correspondientes para mantener la trazabilidad entre gestión y desarrollo.
